## CENTRALIZED TASK SYSTEM (ALL AGENTS)
Multi-agent task board at `C:\Users\Carbon\projects\tasks.bat` — all agents check here:

```
tasks list              # Check for work (poll every 30s)
tasks claim <id> <name> # Claim a task so no one else takes it
tasks done <id>         # Mark complete
tasks add "title" "desc" -tag "area"  # Add new work
tasks mine <name>       # What I'm working on
tasks agents            # See all agent activity
```

**Rules:**
1. Poll `tasks list` every 30 seconds for new work
2. Always `claim` before starting work
3. Never create a task without checking `tasks list` first (auto-dedup)
4. `tasks done` when finished, or `tasks fail` if blocked
5. Stale claims (>30 min) auto-reset to pending
